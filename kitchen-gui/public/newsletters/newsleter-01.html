We now are a team

Meteor Kitchen was started as a personal hobby project in 2013 while Meteor was 0.5 beta. Meteor changed a lot since then, and it was almost a nightmare to keep the code generator up to date. However, somehow Meteor Kitchen survived all the "earthquakes" and today we have both CLI and GUI with 5300+ registered users.



Today, almost 4 years later, we are happy to announce that we have grown to a 5 member team, and now you can expect a lot of improvements and new features like:

• Align generated code with Meteor's best practices
• Improve visual UI designer
• Write better docs and tutorials

...and more...

We now help you to make money

In our projects we always experience a strong Meteor Community but on existing code hire platforms Meteors still is a niche. This is why we have decided to leverage our Kitchen Kommunity to connect those running projects with those happily hacking away.



Meteor developer marketplace


Are you looking for a job?

Please login into Meteor Kitchen, then go to your profile settings and set "I am available for hire" switch on.
Companies who are looking for developer will be able to contact you.



Are you looking for Meteor developers?

Please login into Meteor Kitchen, then goto "Developers" page, and at the top of the page there is filter "Available for Hire". You'll see list of all developers who are searching for a job, you can see their profile with skills and you can contact them.
